基础知识：

  • 云帐号是什么，有什么用

云帐号是您使用云服务所有服务时所需要的用户系统，通过注册和登录云帐号，您可以：

  • • 使用云服务，主题市场，应用商店，MIUI论坛等以MIUI为中心的衍生产品。

  • • 登录米聊，联络老朋友，认识新朋友。

  • • 登录米吧，认识一群爱唱歌爱漂亮的朋友。

  • • 登录云服务官网和云服务社区，在线购物，结识好友，反馈意见问题。

  • • 登录多看阅读，体验专业的排版设计，全新的阅读交互，“多看书，多交友”。

如果您使用过上述任何一个产品，即表示您已经拥有了云帐号，您的帐号名和密码可以用来登录其他任一云服务。您注册时绑定的邮箱、生成的云服务ID、以及您之后绑定的密保邮箱都可以作为云帐号名。

还没有云帐号吗？赶快注册一个吧！

登录帐号：

  • 为何无法正常登录 短信无法获取

无法正常登录，原因有以下几种可能：

  • 有违规操作

    您曾经在登录帐号后，执行了一些违规操作，如发广告贴等。帐号被暂时冻结。您可以：

    - 等待一段时间后重试。

    - 联系客服

  • 多次输入错误密码

    您在登录帐号时，多次输入错误密码，导致帐号被暂时冻结。您可以：

    - 等待1小时后重试。

邮箱邮件无法获取，原因有以下几种可能：

  • 被运营商屏蔽

    您曾将此类通知短信向运营商投诉为垃圾邮件，使得邮件被运营商屏蔽

    - 需要您联系运营商解决此问题


  • 邮箱黑名单设置

    您曾经在邮箱邮件中设置了比较严苛拦截的拦截规则

    - 查看邮箱黑名单设置

    - 以MIUIv5为例，进入拨号->菜单->骚扰拦截->设置，分别查看拦截模式与黑白明单是否能允许云帐号登录邮件的到达


  • 浏览器隐私设置

    浏览器设置安全等级过高或者设置了阻止“第三方Cookie和网站数据”的设置

    - 重新设置浏览器安全等级或者关闭浏览器“阻止第三方Cookie”的设置

    - 以Chrome 35为例，chrome->设置->隐私设置->内容设置


  • 邮件发送超过当天限制

    为避免邮件轰炸，以及资源浪费，每个云帐号最多每天发送十条邮件，每个邮箱账号每天最多发送十条短信。

    - 当天不能再登录需要验证邮箱的服务。

  • 网络延时

    由于网络或地域问题，服务器发出的验证信息有可能存在一定延时

    - 耐心等待，或读秒完成后重试

注册帐号：

  • 为何注册时需要绑定邮箱
  • 为何有些情况下无法收到验证邮件
  • 为何无法正常使用邮箱注册
  • 您必须年满多少岁才能注册云帐号

您注册云帐号后，可以通过云帐号使用云服务、云币中心、米聊等多项云服务。为了维护您帐号安全性，在注册过程中我们要求您提供部分个人资料，其中包括邮箱。他们会成为您帐号的绑定密保，主要用于找回密码和修改密保，保护帐号安全：

  • 邮箱地址

    电子邮件地址具有长期固定的特征，不易丢失或变更。提供此地址，有助于保护您帐号安全和找回密码，同时也为小米服务发送通知提供媒介。并且我们可以通过验证您的邮箱地址阻止对您帐号的非正常访问。

无法收到验证邮件，原因有以下几种可能：

  • 输入地址错误

    您可能在输入邮箱时，输入错误，验证邮件未能发送到您的地址。您可以：

    - 检查手机邮箱是否填写正确，返回并重新输入

  • 我们的服务器发送有延迟

    由于网络或地域问题，服务器发送验证信息受通讯网络讯号和运营商支持影响，可能存在一定延时。您可以：

    - 点击“重新发送”，服务器再发送一遍验证邮件。

    - 联系客服

无法正常使用邮箱注册，原因有以下几种可能：

  • 您注册时使用的邮箱已经绑定另一云帐号。为保障帐号安全，一个邮箱只能绑定一个云帐号。您可以：

    - 更换未绑定云账户的邮箱重新注册。

    - 使用已绑定云帐号的邮箱直接登录。

  • 您多次使用同一设备电脑注册云帐号，导致注册功能或需要注册的邮箱被暂时封禁。为防止多次注册无效帐号的不良注册行为，我们为同一设备注册帐号次数设置了上限，用户使用同一设备注册帐号的次数到达上限后，注册功能被暂时封禁。您可以：

    - 等待一段时间后重试。

    - 联系客服。

注册云帐号的年龄限制

要符合云帐号的注册资格，您必须至少年满13岁

密保设置：

  • 密保是什么，有什么用
  • 为何绑定或修改密保（邮箱）时,需要进行安全验证
  • 为何绑定密保（邮箱）时，提示已经绑定了其他帐号
  • 如何解除密保（邮箱）的绑定
  • 绑定的密保（邮箱）不可用了怎么办

保护帐号安全的相关工具称作密保，顾名思义也就是密码保护。您注册时绑定的邮箱，以及注册之后新绑定的邮箱都属于密保工具。

忘记密码时可以通过您绑定的邮箱找回密码。在执行影响帐号安全的相关操作时，我们也会通过验证密保来确认用户身份。设置并完善密保信息有助于保护您帐号安全和找回密码，同时也为云服务发送通知提供媒介。

还没有绑定密保邮箱吗？快来完善密保信息吧！

相关问题：- 为何注册时需要绑定邮箱

在登录帐号修改密保时，对原密保进行安全验证，可防止别有用心的人侵用您的帐号。

由于密保邮箱可以用来找回密码、恢复帐号，在帐号安全的保障中扮演极其重要的角色。密保信息（邮箱）的安全直接影响到帐号的安全，所以在修改密保时，我们通过发送验证邮件的方式，确认执行修改操作的是用户本人。

具体步骤：
修改密保 ——> 云帐号安全验证 ——> 设置新密保 ——> 完成

一般用户只有一个邮箱地址，由于云帐号牵涉到实体交易，如充值支付、购买商品等，我们需要对云帐号用户的唯一性提出一定要求。为避免由于密保丢失或被多人使用导致的帐号财产损失，一个邮箱只能绑定一个云帐号。

您需要绑定的邮箱可能之前已经与另一云帐号绑定，所以无法与当前云帐号绑定。

您可以：

- 更换未绑定小米账户的邮箱重新绑定。

- 使用已绑定云帐号的邮箱直接登录。

相关问题：- 为何无法正常使用邮箱注册

设置并完善密保信息有助于保护您帐号安全和找回密码，同时也为云服务发送通知提供媒介。我们鼓励绑定完善的密保信息，在绑定密保后，暂不提供解除绑定的功能。

您需要绑定的邮箱可能之前已经与另一云帐号绑定，所以无法与当前云帐号绑定。

如遇特殊情况需解除绑定时，您可以：

- 修改密保信息，新密保覆盖旧密保

- 联系客服

由于手机和邮箱的丢失或更换，导致密保工具不可用时，您可以通过验证老密保或验证其他密保的方式更改密保信息。

如：更改绑定手机，需要验证原绑定邮箱。

如果所有密保信息都不可用，无法验证，您可以通过申请人工审核的方式修改密保信息。

删除帐号：

  • 删除帐号意味着什么
  • 如何删除帐号

删除帐号意味着帐号所有相关信息将永久丢失，包括用户名、密码、米币、交易记录，论坛帖，云同步信息等。帐号将不复存在。

删除后无法再找回帐号，是危险操作，需慎重！

由于删除帐号属于危险操作，后果严重，我们不提供由用户控制的删除功能。

如遇特殊情况需删除帐号时，您可以：

- 联系客服

云服务：

  • 为何云服务中关联的邮箱账号不是绑定的邮箱账号
  • 为何米聊中关联的多个联系方式（邮箱）没有显示在云帐号中

帐号绑定邮箱号：密保邮箱，只能唯一。密保邮箱可能是您注册时绑定的或注册后绑定的，用于登陆和找回密码。

云服务中关联的邮箱号：当前登录在线并使用云服务的电脑设备，我们允许多个移动设备登录同一云帐号，所以云服务中关联的邮箱号可以为多个。

两者本质上没有关联，所以可能会不同。某些情况下也可能相同，如：

云帐号的密保邮箱同时登录了同一云帐号并开启了云同步，也会成为云服务的关联手机设备。

相关问题：- 密保是什么，有什么用

云聊关联的联系方式是没有经过验证的关联信息，用户可以填写多个，但不能作为密保工具用于找回密码和修改密保。

云帐号中可见的绑定邮箱是云帐号的密保工具，经过验证并具有唯一性，用于保护帐号安全，找回密码和修改密保。

两者本质上不同，绑定密保是帐号的重要组成部分，所以可以在云账户中进行管理。云聊中关联的联系方式作为云聊的特定信息，暂不出现在云帐号中。
